# Privacy Policy for Cereal Discord Bot

**Last Updated: January 23, 2026**

## 1. Introduction

Welcome to Cereal ("we," "us," or "our"), a Discord bot designed to enhance your Discord server experience with moderation, games, fun commands, and utility features. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our bot.

## 2. Information We Collect

### 2.1 Discord User Data
We collect the following information from Discord:
- **User ID**: Unique identifier for Discord users
- **Username and Discriminator**: Your Discord display name
- **Server/Guild Information**: Server IDs and names where the bot is used
- **Role Information**: User roles within servers (for moderation features)

### 2.2 Usage Data
- **Command Usage**: Which commands you use and when
- **Server Statistics**: Number of servers, users, and channels
- **Bot Performance Metrics**: Latency, uptime, and error logs

### 2.3 External API Data
Our bot integrates with third-party services:
- **OpenWeatherMap API**: Location data for weather commands
- **Reddit API**: Content for meme and fun commands

## 3. How We Use Your Information

We use the collected information to:
- Provide and maintain bot functionality
- Process commands and respond to user interactions
- Monitor bot performance and troubleshoot issues
- Improve bot features and user experience
- Generate anonymous usage statistics

## 4. Data Storage and Security

### 4.1 Data Storage
- All data is stored locally in a SQLite database
- No personal data is transmitted to external servers except for API calls
- Data is retained only as long as necessary for bot functionality

### 4.2 Data Security
- We implement appropriate security measures to protect your data
- Access to data is limited to bot operations only
- No data is encrypted at rest (SQLite limitation)

## 5. Data Sharing and Disclosure

We do **NOT**:
- Sell, trade, or rent your personal information to third parties
- Share user data with advertisers or marketing companies
- Transmit data to external servers except for required API functionality

Data may be shared only in the following circumstances:
- **Legal Requirements**: When required by law or legal process
- **Bot Operation**: With Discord's API for bot functionality
- **Third-Party APIs**: As necessary for command functionality (weather, memes)

## 6. Third-Party Services

Our bot uses the following third-party services:
- **Discord API**: Governed by [Discord's Privacy Policy](https://discord.com/privacy)
- **OpenWeatherMap**: Governed by [OpenWeatherMap Privacy Policy](https://openweathermap.org/privacy-policy)
- **Reddit API**: Governed by [Reddit's Privacy Policy](https://www.reddit.com/policies/privacy-policy)

## 7. Your Rights and Choices

### 7.1 Data Access
You can request information about what data we store about you by contacting us.

### 7.2 Data Deletion
- You can remove the bot from your server at any time
- Contact us to request deletion of any stored data
- Some anonymized usage statistics may be retained for analytics

### 7.3 Opt-out
- Disable specific bot features by not using certain commands
- Remove the bot from your server to stop all data collection

## 8. Children's Privacy

Our bot is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

## 9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by:
- Updating the "Last Updated" date
- Posting the updated policy in our GitHub repository
- Announcing changes in bot status or relevant Discord channels
